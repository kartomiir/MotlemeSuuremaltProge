# MotlemeSuuremaltProge
Mõtleme Suuremalt program, both versions. Suremalt is the long one, SuremaltV2 does the same but is cut into multiple files (xfirst, xsecond, xother_options and such)

Mõtleme Suuremalt programm, mõlemad versioonid. Suremalt on pikk proge, SuremaltV2 teeb lõppude lõpuks sama aga on tükeldatud (zesimeme. zteine, zmuud_valikud jne)

Letters in file names (x, y, z) are so that everything is sorted alphabetically.

Tähed faili nimedes (x, y, z) on selle jaoks, et kõik oleks tähestiku järgi sorteeritud.

Program might not work because putting them on github might've changed a few things

Proge võibolla ei tööta sest githubi panemine võis asjad sassi ajada
